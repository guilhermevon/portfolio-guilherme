## 📁 Estrutura

```
📁 imagens/            → Imagens do portfólio e projetos  
📄 index.html          → Página principal  
🎨 styles.css          → Estilo do site  
📄 Currículo.pdf       → Download direto do currículo  
📜 script embutido     → Lógica JS de animações e modais
```
